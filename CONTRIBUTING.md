# Contributing to Letterboxd Randomizer

Thank you for your interest in contributing! We welcome all kinds of contributions, whether it's fixing bugs, adding features, or improving documentation.

## Getting Started

1. **Fork the repository** to your own GitHub account.
2. **Clone the repository** to your local machine:
   ```bash
   git clone https://github.com/basithladoo/letterboxdrandomizer.git
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```

## Making Changes

1. **Create a new branch** for your feature or bugfix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. **Make your changes**. Please ensure your code follows the existing style and is well-tested.
3. **Commit your changes** with a clear and descriptive message:
   ```bash
   git commit -m "Add feature: slot machine sound effects"
   ```
4. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
5. **Open a Pull Request** (PR) against the `main` branch of the original repository.

## Feedback and Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/basithladoo/letterboxdrandomizer/issues) before starting work. This helps avoid duplicate efforts and ensures the feature aligns with the project goals.
