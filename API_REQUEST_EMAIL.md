Hi Letterboxd Team,

I am writing to request API access for my project, **Letterboxd Randomizer**. 

**Project Description:**
Letterboxd Randomizer is a fun, retro-styled web application that helps users decide what to watch next by picking a random film from their public watchlist. It features a slot-machine style animation to make the discovery process engaging.

**Intended Use:**
Currently, the app relies on public scraping and CORS proxies, which can be unreliable and slow. We would like to use the official API to fetch user watchlists more efficiently and provide a better experience for the Letterboxd community.
